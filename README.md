> **카카오 비즈 메시지(알림톡/상담톡/친구톡) 발송 데이터 전처리 및 딜러사 종합 지표 분석 자동화 파이프라인**

본 프로젝트는 카카오 비즈 메시지 사업 지원팀의 핵심 실무인 **'대용량 메시지 발송 데이터 정리'**, **'딜러사별 주요 지표(매출, 성공률) 분석'**, 그리고 '반복적인 엑셀 취합 및 리포트 작성 업무의 자동화'를 파이썬과 `Pandas` 라이브러리를 활용해 구현한 포트폴리오입니다.

* `import pandas pd.py`: 여러 폴더에 흩어진 복수의 딜러사 주간 리포트(Excel)를 하나의 마스터 데이터로 병합하고 결측치를 처리하는 취합 자동화 스크립트

## 📈 주요 기능 및 구현 내용

### 1. 다중 엑셀 데이터 자동 취합 (`import pandas pd.py`)
* 무작위로 인입되는 여러 딜러사의 주간 엑셀 리포트 파일들을 `glob` 라이브러리로 일괄 탐색합니다.
* 반복문과 `pd.concat`을 통해 수십 개의 파일을 단 하나의 마스터 데이터프레임으로 자동 병합하여 휴먼 에러를 방지하고 리소스를 대폭 절감합니다.

### 2. 실무 밀착형 데이터 전처리 및 정제 (Data Cleaning)
* 현업에서 빈번히 발생하는 데이터 누락 상황을 가정하여, '요청건수' 등의 결측치(`NaN`)를 찾아 분석 오류가 나지 않도록 `fillna(0)` 기법으로 정제합니다.
* 파일 내 한글 깨짐 현상을 원천 차단하기 위해 바이트 표식(BOM)을 포함한 `utf-8-sig` 인코딩 방식을 적용하여, 현업 스태프들이 엑셀로 즉시 열람할 수 있도록 범용성을 높였습니다.

### 3. 비즈 메시지 지표 분석 및 파생변수 생성
* `groupby` 연산을 통해 흩어진 로그 데이터로부터 각 딜러사별 **총 요청건수, 총 성공건수, 총 매출액**을 계산합니다.
* 딜러사 관리의 핵심 지표인 **[발송성공률(%)]** 파생변수를 수식화하여 추가하고, 소수점 둘째 자리로 반올림(`round(2)`)하여 시각적 가독성을 확보했습니다.

### 4. 메시지 유형별 피벗 테이블 보고서 자동화
* 엑셀의 복잡한 피벗 테이블 기능을 판다스의 `pd.pivot_table` 코드로 구현했습니다.
* 알림톡, 상담톡, 친구톡(텍스트/이미지), 브랜드링크 등 각 메시지 유형별 발송 현황을 딜러사별 매트릭스 형태로 한눈에 파악할 수 있어 우수 딜러사 및 집중 관리 대상 딜러사를 빠르게 선별합니다.
